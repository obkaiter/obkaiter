### RemnaSuper

[RemnaSuper](https://github.com/obkaiter/RemnaSuper) - интерактивное Shell-меню для обслуживания RemnaNode. Проект помогает быстрее выполнять рутинные серверные операции: работать с логами, настраивать logrotate, управлять geofiles, перезапускать сервисы, смотреть ошибки Xray и запускать вспомогательные команды для RemnaWave-инфраструктуры.
